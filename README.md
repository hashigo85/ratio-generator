# UMvC3 Ratio Chart Generator

オリジナルの「レシオ表」をブラウザ上で簡単に作成・共有・データ提出できるWebアプリケーションです。
オンライン大会の運営や、コミュニティ内でのチーム編成の共有に最適化されています。

## 主な機能 (Features)

キャラクターのマスをタップ／クリックして、ドロップダウンから点数を選ぶだけで簡単にレシオを割り当てられます。
表の上部スペースをクリックすることで、大会名やプレイヤー名など、好きなタイトルを入力可能。長さに応じて自動でサイズが調整されます。
操作中のマスが白枠でハイライトされ、PCでもスマートフォンでも迷わず操作できます。
生成した画像は端末に自動保存されるだけでなく、スマートフォンのネイティブ機能（Web Share API）を使ってワンタップで X (Twitter) へ画像付きでシェアできます。
「レシオ表を作成」ボタンを押すと、画像データをして保存できます。

こちらのURLから実際にツールを使用できます。
👉 **https://hashigo85.github.io/ratio-generator/**

## 使い方 (How to Use)

1. 上記のURLにアクセスします。
2. 上部の黒いタイトルスペースをクリックし、任意のタイトルを入力します。
3. 各キャラクターのマスをクリックし、リストから割り当てたい点数（1〜8、または ×）を選択します。
4. **「レシオ表を作成(Create Ratio Chart)」** ボタンを押します。
5. 完成した画像が自動でダウンロードされます。
6. **「共有(Share)」** ボタンを押すと、X等で #アルカプレシオ のハッシュタグと共に画像を投稿できます。

## 🛠 開発者向けセットアップ (Setup for Developers)

このリポジトリをフォークして独自のジェネレーターを構築する場合は、以下の手順に従ってください。

### 1. ファイルの配置
* `index.html` （メインプログラム）
* `base_chart.png` （ベースとなる背景画像。7行×8列のフォーマットである必要があります）
上記2つのファイルを同じディレクトリに配置してください。
